# 🧠 Turing Machine Simulator

A web-based simulation of a Turing Machine built using HTML, CSS, and JavaScript. This project demonstrates how a Turing Machine processes input strings step-by-step, including state transitions, tape movement, and halting conditions.

---

## 🚀 Features

* Step-by-step execution of Turing Machine
* Accept and Reject state detection
* Infinite loop detection (bounded steps)
* Visual representation of tape and head movement
* User input for custom strings

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
